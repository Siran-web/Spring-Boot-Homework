It is used to express request mapping , request Input, exception handling and more.

@Controller
@RestController (@Controller + @ResponseBody)


We can use @RequestMapping to map request to controllers method.

Short cuts method od @RequestMapping
@GetMapping
@PostMapping
@DeleteMapping
@PutMapping
@PatchMapping

@PathVariable - When data is essential part for URL path that identifies resource
@RequestParams - When data is optional part of URL that is use for filtration, 
                sorting or other modification.

@RequestBody - It is used to bind HTTP request body with java object.

@Entity - This annotation marked a class as persistence class. that map object with table.

---------> Handling annotations <--------
@ExceptionHandler - It use to handle exception that occur in controllers.
@RestControllerAdvices - It is used to handle global exceptions.
